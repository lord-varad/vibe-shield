# VibeShield: Silent Security for AI Agents

VibeShield is a universal security layer that sits between an AI agent and the user. It ensures that any code generated by the AI is secure and free of sensitive leaks before you ever see it.

## What it does
VibeShield monitors the AI's output in the background. If the AI writes insecure code—such as SQL injection, XSS, or hardcoded credentials—VibeShield catches the error and tells the AI to fix it quietly. The AI then rewrites the code correctly, so you only receive the secure version.

## Visual Feedback
Whenever VibeShield is actively scanning in the background, a small `[✧]` symbol will briefly appear in your terminal to let you know the security layer is active.

## Features
- **Silent Fixes**: Uses `semgrep` to automatically identify and request fixes for security vulnerabilities.
- **Secret Protection**: Uses `trufflehog` to stop the AI from accidentally leaking API keys, tokens, or credentials.
- **Progress Tracking**: Use the `show_shield_report` tool to see a detailed report of total scans and how many silent fixes have been applied.

## Easy Setup
To automatically configure VibeShield for Claude, Gemini, and Cursor, run:
```bash
python install_shield.py
```

## Manual Setup
If you prefer manual configuration, add the following block to your client's `mcpServers` settings:

```json
{
  "mcpServers": {
    "vibeshield": {
      "command": "python3",
      "args": ["/absolute/path/to/vibe-shield/server.py"]
    }
  }
}
```

## Requirements
- **Python 3.10+**
- **semgrep**: `pip install semgrep`
- **trufflehog**: `pip install trufflehog`
